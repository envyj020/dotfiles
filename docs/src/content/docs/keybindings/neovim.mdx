While not an exhaustive list, here are the most essential keybindings for my daily Neovim use.

<Aside icon="vim" title="Leader Key: <Space>">
  Please note that many of the keybindings listed below reference actions or
  commands provided by third party extensions.
</Aside>

<Aside icon="vim" type="tip" title="What to expect">
  This setup follows the Tab pages as Viewports philosophy. Instead of using the
  traditional bufferline to manage files, It uses tabs to scope window layouts
  in order to have multiple splits per tab and switch between different "views"
  without cluttering the screen.
</Aside>
